# 📊 CSV Data Visualization Web App

An interactive data visualization web application built using **Streamlit**, designed to simplify exploratory data analysis by allowing users to upload, inspect, and visualize CSV datasets in real time.

---

## 🚀 Overview

This project enables users to quickly transform raw CSV data into meaningful visual insights without writing complex code. It provides an intuitive interface for selecting variables and generating charts dynamically.

---

## ✨ Key Features

* 📁 **CSV Upload Support** – Upload and process datasets instantly
* 👀 **Data Preview** – View structured tabular data
* 🎯 **Dynamic Axis Selection** – Choose X and Y axes interactively
* 📊 **Interactive Visualizations**

  * Bar Charts
  * Pie Charts
* ⚡ **Real-Time Updates** – Charts update instantly

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly Express

---

## 🧩 Architecture

1. **Input Layer**

   * File uploader (Streamlit)

2. **Processing Layer**

   * Data handling using Pandas
   * Detects numeric & categorical columns

3. **Visualization Layer**

   * Chart rendering using Plotly
